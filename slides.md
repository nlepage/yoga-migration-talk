# GraphQL <img src="/graphql.png" class="inline w-12 ml-2">

<div class="flex gap-10">

```graphql
query getStudents {
  students {
    id, firstName, lastName
    grades {
      id, grade
      subject {
        id, name
      }
    }
  }
}
```

```graphql
type Student {
  id: ID!
  firstName: String!
  lastName: String!
  grades: [Grade!]!
}

type Grade {
  id: ID!
  grade: Float!
  subject: Subject!
}
```
